2. Microdata Detection

Detects itemscope, itemtype, and itemprop attributes.

// Detects microdata patterns const microdataMatches = html.match(/itemscope[^>]*itemtype=["']([^"']+)["'][^>]*>/gi); // Looks for schema.org/Article, schema.org/Person, etc. // Also detects itemprop patterns for article properties

3. RDFa Detection

Detects typeof and property attributes used in RDFa.

// Detects RDFa patterns const typeofMatches = html.match(/typeof=["']([^"']+)["'][^>]*>/gi); // Looks for schema:Article, schema:Person, etc. // Also detects property patterns for schema properties

4. HTML Structure Patterns

Detects common HTML patterns that indicate schema-worthy content.

// Detects article-like structures const articlePatterns = [ /<article[^>]*>/gi, /<main[^>]*>/gi, /<section[^>]*class=["'][^"']*article[^"']*["'][^>]*>/gi ]; // Detects author-like structures // Detects FAQ-like structures

5. Meta Tag Analysis

Analyzes Open Graph, Twitter Cards, and other meta tags that indicate structured content.

// Detects Open Graph article type const ogArticle = html.match(/<meta[^>]*property=["']og:type["'][^>]*content=["']article["'][^>]*>/gi); // Detects Twitter Cards // Detects author meta tags
