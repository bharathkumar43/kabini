Problem: The `isApplying` state was not being set in `applySingleSuggestion`, causing the useEffect to run and call `applySuggestionsWithDOM` with all 8 suggestions.

Evidence: The call stack shows that `comparePages` is being called, which means the useEffect is not being prevented from running.

Root Cause: Missing `setIsApplying(true)` and `setIsApplying(false)` calls in `applySingleSuggestion` function.
